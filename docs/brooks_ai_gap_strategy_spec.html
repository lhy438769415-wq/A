📖 量化条件与交易规格

1. 突破缺口的量化判定 (Breakout Gap)

识别高权重“结构性突破缺口”的入场底线：

参数/条件 量化数学公式 PA 物理逻辑
回溯窗口 ($N$) 周线: $N = 60$ 周，日线: $N = 60$ 天 确立一个足够有宏观权重的阻力区间。
微观价格抬升 $high_t > high_{t-1} \text{ 且 } low_t > low_{t-1}$ 保证当根突破K线具有极强的向上物理动能。
宏观防守线 $Gap\_Floor = \max(high_{t-N} \dots high_{t-2})$ 过去 $N$ 根K线的绝对最高价，作为“阻力转支撑”的底线。
物理缺口确立 $low_t > Gap\_Floor - 0.001$ 防击穿铁律：突破日最低点也必须完全悬空于防守线之上。
起涨波段大底 $Prior\_Swing\_Low = \min(low_{t-N} \dots low_{t-2})$ 本轮攻势发起前的“深渊底”，作为镜像测距的起点。
初始缺口上沿 $Breakout\_Low = low_t$ 突破日 K 线的最低价，标志着初始缺口的物理上沿。

2. 核心买入触发模式

模式 A：宏观结构缺口与回调反转 (Structural Gap Reversal)

寻找强势突破后，深度回踩但不破防线，且重新止跌启动的二波趋势机会。

  1. 回调时间窗限制：突破后回调持续时间必须满足： $2 \le \text{回调根数} \le 40$。
  2. 缺口存活监控 (Surviving)：自突破以来的最低价满足： $Min\_Low_t > Gap\_Floor - 0.001$。
  3. 止跌与反转信号确认：
    • 下行停滞（H1 止跌）：$low_t \ge low_{t-1}$（Lower Low 终结）。
    • 多头重新接管：收阳线，且收盘创前日新高：
      $high_t > high_{t-1} \text{ 且 } close_t > open_t \text{ 且 } close_t > close_{t-1}$
  4. 高潮规避限制 (MM Not Reached)：自突破以来的最高点，不得已经打满翻倍测距目标 $TP$，防止追高强弩之末的走势。

模式 B：突破缺口 + Pinbar 极限测试 (Gap + Pinbar Test)

主力通过一根长下影线（Pinbar）精准下探测试缺口，测试完毕后被多头迅速拉回，是胜率极高的“特种兵”形态。

  1. 回调时间窗限制：同样必须在突破后的 $2 \le \text{回调根数} \le 40$ 之内。
  2. EMA均线贴合度：Pinbar 的收盘价距离 $EMA_{20}$ 必须在 $1.5$ 倍 $ATR$ 以内，保证处于密集交易区：
    $|close_t - EMA_{20, t}| \le 1.5 \times ATR_t$
  3. Pinbar 经典形态判定：
    • 下影线比例：$\text{下影线占比} = \frac{\min(open_t, close_t) - low_t}{high_t - low_t} \ge 0.40$
    • 收盘位置：$\frac{close_t - low_t}{high_t - low_t} \ge 0.50$ （收盘在 K 线的中位偏上方）
  4. 缺口极限测试判定：Pinbar 的下影线必须向下探测缺口空间。满足以下之一：
    • 条件 a (探入)：Pinbar 最低点探入此前回调期间的最低点下方： $low_t \le Min\_Low_{t-1}$。
    • 条件 b (接近)：最低点与缺口下沿的间距不大于 $2$ 倍 ATR： $low_t - Gap\_Floor \le 2.0 \times ATR_t$。
  5. 缺口未穿透校验：收盘价必须在 $Gap\_Floor$ 之上，且最低点 $low_t$ 同样不能击穿 $Gap\_Floor - 0.001$。

3. 交易设置与生命周期过滤

执行挂单与生命周期的强力风控机制：

  • 买入入场价 (Entry)：触发信号当天 K 线的最高价 $high_{sig}$（次周期以 Buy Stop 限价单 挂单）。
  • 止损设置 (Stop Loss)：
    • 方案 A（精细止损）：$SL = Gap\_Floor$ （挂在突破下沿支撑，盈亏比极佳）
    • 方案 B（宽幅止损）：$SL = Prior\_Swing\_Low$ （挂在起涨波段底，防爆仓率高）
  • 非对称测距止盈 (Take Profit - Measured Move)：
    • 寻找回调期间的有效缺口上沿 $Gap\_Top$（为信号前整个回调期砸出的最低低点）。
    • 计算缺口中心轴：$Gap\_Mid = \frac{Gap\_Floor + Gap\_Top}{2}$。
    • 计算测距目标：$TP = 2 \times Gap\_Mid - Prior\_Swing\_Low$。
  • 存活生命周期过滤规则 (Life Cycle Rule)：
    • 防守端（被回补死亡）：若未被触发，一旦有任何一根 K 线的最低点跌破了 $Gap\_Floor$（包含信号K线本身），立即撤单失效。
    • 收益端（止盈先达）：若价格未突破 Entry，但先打到了 $TP$ 目标，交易计划直接作废。
    • 时限端（超时失效）：信号发出后超过 30 根 K 线仍未触发，判定为横盘僵尸股，强制抛弃。
