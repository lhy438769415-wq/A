
# 📖 Brooks-AI 量化系统 V8.30 系统说明书

## 1. 系统愿景 (Vision)
本系统专注于 **Al Brooks 价格行为 (Price Action)** 理论在 A 股市场的落地。通过精简、严苛的过滤机制，寻找具备高数学期望（Expectancy）的交易机会。

---

## 2. 核心交易策略 (Core Strategies)
系统目前仅保留两套经过深度验证的逻辑：

### 🦅 MTR 反转策略 (Master V29.5)
- **核心逻辑**: 捕捉大级别趋势末端的重大趋势反转（Major Trend Reversal）。
- **数学属性**: 
  - 期望值: **+0.25 R**
  - 核心胜率 (2R): 约 30%
  - 心理胜率 (1R): 约 50.6%
- **操作重点**: 触达 1R 后建议减仓 50% 并将止损移至保本，锁定利润博取 2R 或更远的 Measured Move。

### ⚡ 3K 动能策略 (EX-Pulse)
- **核心逻辑**: 识别连续三根强实体阳线构成的微观通道爆发。
- **数学属性**: 
  - 期望值: **1:1.5 快速博弈**
- **操作重点**: 止损严格锁定在信号串第一根阳线（K1）的最低点，不纠结，执行效率至上。

---

## 3. 操作手册 (Operation)

### 3.1 猎手扫描 (Hunter)
- **命令**: `python hunter.py` (默认 MTR) 或 `python hunter.py --strategy STRATEGY_3K`
- **建议频率**: 每日 **14:30**（尾盘确立期）或 **18:00**（复盘期）。
- **输出**: 
  1. **极简文本指令**: 含建仓、止损、止盈。
  2. **信号汇总总表**: Markdown 汇总表格。
  3. **专业分析图**: 前 3 只最强信号个股的 K 线图示，标有实时价格刻度。

### 3.2 诊断问股 (Ask)
- **用法**: 在控制台或 GUI 中输入代码。
- **功能**: AI 会强制跳过初筛规则，为您提供当前个股的深度 PA 结构分析。

---

## 4. 绘图体系说明 (Chart Legend)
- **红色烛身**: 阳线 (Bull Bar)
- **绿色烛身**: 阴线 (Bear Bar)
- **绿色水平线 (SL)**: 初始止损，价格破此线即逻辑失效。
- **红色水平线 (TP1)**: 减仓或锁定利润的首选目标（通常为 1R 风险空间）。
- **紫色水平线 (TP2)**: 终极获利或波段止盈位。
- **橙色曲线**: EMA 20 (波段生命线)
- **蓝色曲线**: EMA 60 (趋势主脉络)
