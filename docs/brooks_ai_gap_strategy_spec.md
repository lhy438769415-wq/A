## 1. 基础骨架：突破缺口的量化判定 (Breakout Gap Definition)

在寻找任何缺口交易机会前，必须首先识别出一个合格的“结构性突破缺口”：

| 参数/条件 | 格式 A：Unicode 下标 (纯文本首选) | 格式 B：中括号代码 (编程首选) | PA 理论物理逻辑 |
| :--- | :--- | :--- | :--- |
| **回溯窗口 (N)** | `N = 60` 周 (周线) / `N = 60` 天 (日线) | `N = 60` | 确立一个足够有宏观权重的阻力区间。 |
| **微观价格抬升** | `highₜ > highₜ₋₁ 且 lowₜ > lowₜ₋₁` | `high[t] > high[t-1] 且 low[t] > low[t-1]` | 保证当根突破K线具有极强的向上物理动能。 |
| **宏观防守线** | `Gap_Floor = max(highₜ₋ₙ ... highₜ₋₂)` | `Gap_Floor = max(high[t-N] ... high[t-2])` | 过去 N 根K线的绝对最高价，作为“阻力转支撑”的底线。 |
| **物理缺口确立** | `lowₜ > Gap_Floor - 0.001` | `low[t] > Gap_Floor - 0.001` | **核心防击穿铁律**：突破当天的最低点（下影线）必须完全悬空于防守线之上。 |
| **起涨波段大底** | `Prior_Swing_Low = min(lowₜ₋ₙ ... lowₜ₋₂)` | `Prior_Swing_Low = min(low[t-N] ... low[t-2])` | 本轮攻势发起前的“深渊底”，作为镜像测距的起点。 |
| **初始缺口上沿** | `Breakout_Low = lowₜ` | `Breakout_Low = low[t]` | 突破日K线的最低价，标志着初始缺口的物理上沿。 |

## 2. 核心策略：两大买入触发模式

当突破缺口确立后，市场进入回调期，系统通过以下两种相互独立的模式寻找高胜率的介入点：

### 模式 A：宏观结构缺口与回调反转 (Structural Gap Reversal)
> **适用场景**：寻找强势突破后，深度回踩但不破防线，且重新止跌启动的二波趋势机会。

1. **回调时间窗限制**：突破后回调持续时间必须在 2 到 40 根 K 线之间（即 `2 <= 回调根数 <= 40`）。
2. **缺口存活监控 (Surviving)**：回调期间任何一根 K 线的最低点 `low` 绝不能跌破缺口下沿。即自突破以来的最低价满足：
   * **Unicode 格式**：`Min_Lowₜ > Gap_Floor - 0.001`
   * **代码格式**：`Min_Low[t] > Gap_Floor - 0.001`
3. **止跌与反转信号确认**：
   * **下行停滞（H1 止跌）**：
     * **Unicode 格式**：`lowₜ >= lowₜ₋₁` （代表 Lower Low 终结）
     * **代码格式**：`low[t] >= low[t-1]`
   * **多头重新接管**：当根 K 线的最高点强势穿透前一日最高点，收阳线，且收盘价创前日新高：
     * **Unicode 格式**：`highₜ > highₜ₋₁ 且 closeₜ > openₜ 且 closeₜ > closeₜ₋₁`
     * **代码格式**：`high[t] > high[t-1] 且 close[t] > open[t] 且 close[t] > close[t-1]`
4. **高潮规避限制 (MM Not Reached)**：自突破以来的最高点，不得已经打满翻倍测距目标 `TP`，防止追高强弩之末的走势。

---

### 模式 B：突破缺口 + Pinbar 极限测试 (Gap + Pinbar Test)
> **适用场景**：主力通过一根长下影线（Pinbar）精准下探测试缺口，测试完毕后被多头迅速拉回，是胜率极高的“特种兵”形态。

1. **回调时间窗限制**：同样必须在突破后的 2 到 40 根 K 线之内。
2. **EMA均线贴合度**：Pinbar 的收盘价距离 `EMA20` 的绝对距离必须在一倍半 `ATR` 以内（保证处于交易密集区，避免过度乖离）：
   * **Unicode 格式**：`|closeₜ - EMA20ₜ| <= 1.5 * ATRₜ`
   * **代码格式**：`abs(close[t] - EMA20[t]) <= 1.5 * ATR[t]`
3. **Pinbar 经典形态判定**：
   * **下影线比例**：`下影线占比 = (min(open, close) - low) / (high - low) >= 0.40` （不低于 40%）
   * **收盘位置**：`收盘位置 = (close - low) / (high - low) >= 0.50` （收盘在 K 线的中位偏上方）
4. **缺口极限测试判定**：Pinbar 的下影线必须向下探测缺口空间。满足以下条件之一：
   * **条件 a (探入)**：Pinbar 最低点探入此前回调期间的最低点下方：
     * **Unicode 格式**：`lowₜ <= Min_Lowₜ₋₁`
     * **代码格式**：`low[t] <= Min_Low[t-1]`
   * **条件 b (接近)**：最低点虽未探入上沿，但与缺口下沿的间距不大于 2 倍 ATR：
     * **Unicode 格式**：`lowₜ - Gap_Floor <= 2.0 * ATRₜ`
     * **代码格式**：`low[t] - Gap_Floor <= 2.0 * ATR[t]`
5. **缺口未穿透校验**：收盘价必须在 `Gap_Floor` 之上，且最低点 `low` 同样不能击穿 `Gap_Floor - 0.001`。

## 3. 交易设置与生命周期过滤 (Trade Setup & Life Cycle)

### 3.1 入场与出场参数设置

* **买入入场价 (Entry)**：触发信号当天 K 线的最高价（次日/次周以 **Buy Stop 限价单** 挂单，一旦突破最高价则顺势追入）。
* **止损设置 (Stop Loss)**：
  * **方案 A（精细止损）**：`SL = Gap_Floor` （挂在突破下沿支撑，盈亏比极佳）
  * **方案 B（宽幅止损）**：`SL = Prior_Swing_Low` （挂在起涨波段底，防爆仓率高）
* **非对称测距止盈 (Take Profit - Measured Move)**：
  * 首先寻找缺口回调期间的**真实缺口上沿**：`Gap_Top`（为信号前整个回调期砸出的最低低点）。
  * 计算缺口中心轴：`Gap_Mid = (Gap_Floor + Gap_Top) / 2`。
  * 根据起涨点计算非对称镜像目标：
    `TP = 2 * Gap_Mid - Prior_Swing_Low`

### 3.2 存活生命周期过滤规则 (Life Cycle Rule)
* **防守端（被回补死亡）**：在信号发出后，若 Buy Stop 尚未被触发，一旦有任何一根 K 线的最低点跌破了 `Gap_Floor`（包括信号K线本身），判定**缺口已被完全回填**，立即撤单失效。
* **收益端（止盈先达）**：在信号发出后，如果价格还没突破 Entry 触发买单，但先打到了 `TP` 目标，交易计划直接作废。
* **时限端（超时失效）**：信号发出后超过 **30** 根 K 线仍未触发或结束，判定为横盘僵尸股，强制抛弃。

## 4. V9.0 四因子积分评级体系 (EV Score)

系统会对所有存活的 Gap 信号进行因子积分累加，从而将信号划分为不同的预期级别：

### 4.1 积分规则
* **因子 1：回调周期 (pb_bars)**
  * 回调周/天数 $\le 4$ 根（快速回踩，强多头）：**$+2$ 分**
  * 回调周/天数 $> 7$ 根（拖延回调，动能流失）：**$-2$ 分**
* **因子 2：缺口宽度百分比 (gap_size_pct)**
  * 缺口宽度百分比：`Gap_Size% = (Gap_Top - Gap_Floor) / Gap_Floor * 100`
  * 宽缺口（宽度 $> 7\%$，安全垫厚）：**$+2$ 分**
  * 窄缺口（宽度 $< 3\%$，阻力薄弱）：**$-1$ 分**
* **因子 3：信号K线收盘质量 (sig_quality)**
  * 收盘极为强势（`sig_quality > 0.8`）：**$+1$ 分**
  * 收盘软弱或十字星（`sig_quality < 0.5`）：**$-1$ 分**
* **因子 4：回调阴线连阴数 (pb_consec_bear)**
  * 回调期连续收阴K线 $\ge 3$ 根（抛压过重）：**$-1$ 分**
* **时间横盘惩罚 (bars_passed)**
  * 信号触发后，横盘等待超过 10 根K线还没爆发：**$-2$ 分**
  * 信号触发后，横盘等待超过 5 根K线还没爆发：**$-1$ 分**

### 4.2 评级映射表

| 总积分 (EV Score) | 评级标签 | 数学期望与交易决策 |
| :--- | :--- | :--- |
| **$\ge 3$ 分** | **🌟🌟 极品 (A+)** | 历史回测盈亏比极优，心理胜率与胜率双高，**核心狙击标的**。 |
| **$2$ 分** | **🌟 高预期 (A)** | 优质结构信号，建议积极挂单。 |
| **$0 \sim 1$ 分** | **👍 常态 (B)** | 标准波段形态，可正常配置。 |
| **$-2 \sim -1$ 分**| **⚠️ 低预期 (C)** | 结构偏弱或洗盘时间过长，只建议极轻仓或观望。 |
| **$< -2$ 分** | **💀 毒性 (D)** | **一律拦截过滤**，不予发信。 |
